Comment utiliser le portail fournisseurs de PwC

Copier le lien Link copied to clipboard

Tous les achats de PwC font obligatoirement l’objet de commandes, et chaque commande comporte notamment la désignation, la quantité, le prix, le délai de livraison (ou de réalisation de la prestation), les délais et mode de règlement.

A quoi sert-il ?

Un portail fournisseurs gratuit et convivial pour :

  • gérer les informations de votre société ;
  • créer votre catalogue en ligne ;
  • recevoir et accuser réception de nos bons de commande ;
  • saisir et envoyer vos factures directement depuis l’outil, à notre équipe comptable ;
  • obtenir un état sur le traitement de vos factures.

Comment l'utiliser ?

Si vous utilisez déjà le portail pour d'autres clients, vous recevrez une invitation pour vous inscrire à un compte PwC.

Par la suite, vous aurez la possibilité de demander à faire fusionner vos comptes de portail fournisseurs

Pour les fournisseurs n’utilisant pas le portail fournisseurs PwC, vous recevrez une première invitation pour vous inscrire et créer un compte.
L’accès à ce portail vous permet :

  • d’ajouter des interlocuteurs supplémentaires, avec des vues à votre compte (ex: la comptabilité) ;
  • d’actualiser vos coordonnées ;
  • de communiquer avec l’émetteur du bon de commande de PwC ;
  • d’intégrer sa facture, si vous le souhaitez, et suivre le statut du paiement.

Commandes

Les commandes de produits et/ou de services sont réalisées via cet outil.

PwC transmet ses commandes à l’adresse électronique renseignée par ses fournisseurs dans le portail fournisseurs, sous la forme d’un message électronique contenant le bon de commande en pièce jointe au format pdf.

Factures

Vous disposez de trois options pour nous faire parvenir vos factures et avoirs. PwC ayant choisi la piste d'audit fiable avec contrôle documenté et permanent, nous demandons à nos partenaires de privilégier la première option parmi les trois présentées ci-dessous.

Quelle que soit l’option retenue, il est impératif que vos factures comprennent les mentions suivantes :

  • la raison sociale précise de l’entité PwC facturée ;
  • le nom de votre interlocuteur PwC ;
  • le numéro de commande commençant par Z (si vous avez reçu une commande) ;
  • le numéro d’OTP (si vous n’avez pas reçu de commande) à obtenir de votre contact chez PwC.

A défaut du respect de ces règles, nous ne pourrons traiter vos factures et avoirs.

  • Mail
  • Via notre portail de facturation
  • Courrier

Mail

L’envoi de vos factures électroniques par e-mail ici

  • Seules les factures au format pdf seront acceptées (les autres formats tels que Word,  Excel ou Zip  ne pourront être traités).
  • Les factures pdf ne doivent pas être cryptées à l’aide d’un mot de passe.
  • Les factures manuscrites ne sont pas acceptées.

Pour tout autre sujet, y compris d’éventuelles relances, les e-mails doivent être adressés ici en précisant dans l’objet du mail :

  • le nom de l’entité PwC concernée ;
  • le nom de votre société ;
  • l’objet de votre mail  (ex. “PricewaterhouseCoopers Services GIE - Société DUPONT - Changement d’adresse”).

Via notre portail de facturation

Le dépôt de vos factures sur notre portail Coupa (portail fournisseurs) qui vous permet gratuitement de :

  • visualiser l’ensemble des commandes que PwC vous a envoyées ;
  • suivre l’état des réceptions de ces commandes ;
  • déposer vos factures en référence à ces commandes ;
  • suivre le paiement de ces factures ;
  • charger, mettre à jour vos références.
